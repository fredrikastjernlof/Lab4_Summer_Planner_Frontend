#  📌Laboration 4 – Autentisering och säkerhet i webbtjänst

## 🧾 Beskrivning

Detta repository innehåller del 2 av Laboration 4 i kursen Backend-baserad webbutveckling.
I denna del av uppgiften skapas en webbplats som använder Fetch API för att konsumera webbtjänsten som skapades under del 1.

Uppgiften kräver att det ska finnas formulär för att kunna registrera användarkonton, samt att kunna logga in med dessa.

---

## 🎯 Syfte

Syftet med uppgiften är att:

- Kunna skapa funktionalitet för autentisering med registrering av användarkonton samt inloggning.
- Kunna använda JWT's för sessionshantering för att förhindra obehörig åtkomst till resurser.
- Kunna skydda känslig data så som lösenord i databasen.
